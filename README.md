# Personal Portfolio Website

I designed and developed a fully responsive personal portfolio website to showcase my technical skills, projects, certifications, education, and achievements in an interactive and visually appealing way. The portfolio was built from scratch using **HTML5**, **CSS3**, **Bootstrap 5**, and **JavaScript**, with a strong emphasis on modern UI/UX principles and responsive web design.

The website features a clean and elegant layout enhanced with smooth scrolling, animated transitions, and interactive hover effects. I incorporated dynamic mouse-hover animations, engaging visual effects, and responsive components to create an immersive user experience across desktops, tablets, and mobile devices.

To improve the overall appearance and usability, I utilized Bootstrap's grid system and reusable components while customizing the styling with CSS animations, gradients, shadows, and transition effects. Interactive elements such as animated buttons, navigation menus, skill cards, project cards, and social media icons respond smoothly to user interactions, making the website more engaging.

The portfolio includes dedicated sections for my professional introduction, technical skills, projects, education, certifications, achievements, and contact information. It also provides direct access to my GitHub repositories, LinkedIn profile, and resume, allowing recruiters and visitors to explore my work easily.

Building this project strengthened my front-end development skills and gave me practical experience in responsive design, UI/UX development, Bootstrap components, CSS animations, and creating interactive web interfaces. This project reflects my creativity, attention to detail, and passion for developing modern, user-friendly websites.

**Technologies Used:** HTML5, CSS3, Bootstrap 5, JavaScript, Responsive Web Design, CSS Animations, Hover Effects, Smooth Scrolling, and UI/UX Design Principles.
